# WindPower — Slimefun4 风力发电附属插件

为 Slimefun4 设计的完整风力发电科技树插件，添加了 **10 种新材料**、**7 个等级的风力发电机**、**1 台电力造纸机**和 **12 项研究**。

---

## 要求

| 依赖 | 版本 |
|------|------|
| Minecraft | 1.20+ |
| Java | 17+ |
| Slimefun4 | RC-34 或更高 |
| 服务端 | Spigot / Paper / Purpur |

## 安装

1. 将 `WindPower v1.1.1.jar` 放入服务器的 `plugins/` 目录
2. 确保已安装 Slimefun4
3. 重启服务器
4. 在 Slimefun 指南中找到「风力能源」分类

## 风力发电机 (7 级)

所有发电机均使用**自定义头颅纹理**作为 3D 立体外观。

| 等级 | 发电量 | 缓冲 | 高度 | 关键材料 |
|------|--------|------|------|------|
| 基础 | 20 J/s | 无 | Y≥60 | 铁块+碳纤维板+强化叶片 |
| 强化 | 35 J/s | 无 | Y≥60 | 钛锭+基础涡轮 |
| 高级 | 45 J/s | 无 | **无视** | 末影之眼+钻石块+风之水晶 |
| 量子 | 150 J/s | 无 | **无视** | 量子核心+下界合金 |
| MK I | 180 J/s | 1024 J | **无视** | 风暴矩阵+下界之星 |
| MK II | 256 J/s | 2048 J | **无视** | 风暴矩阵+末影水晶+下界之星 |
| MK III | 512 J/s | 4096 J | **无视** | 4×MK II+时空共振核心+下界之星 |

### 发电机制

- **高度要求**：基础/强化需 Y≥60，高级及以上无视高度
- **天空视野**：上方有直接天空视野 → 满功率
- **半遮挡**：部分遮挡 → 功率减半
- **全封闭**：完全 enclosed → 停止发电
- **缓冲容量**：量子以下无缓冲（需即时消耗），MK I 及以上可储电

所有数值均可在 `config.yml` 中调整。

## 造纸机

使用电力将植物材料加工成纸张，支持货运系统自动输入。

| 配方 | 产出 | 耗时 |
|------|------|------|
| 2× 竹子 | 1× 纸 | ~10秒 |
| 1× 甘蔗 | 1× 纸 | ~5秒 |
| 16× 树叶 | 5× 纸 | ~15秒 |

- 图标：制图台
- 耗电：32 J/s
- 支持所有原版树叶类型

## 命令

| 命令 | 说明 | 权限 |
|------|------|------|
| `/windpower` 或 `/wp` | 查看所有发电机信息 | 默认 |

## 配置

所有发电量、高度、容量均可通过 `plugins/WindPower/config.yml` 调整：

```yaml
wind-turbines:
  basic:
    energy-output: 20
    min-height: 60
    capacity: 0
    ignore-height: false
  quantum-mk3:
    energy-output: 512
    min-height: 0
    capacity: 4096
    ignore-height: true

paper-mill:
  energy-consumption: 32
```
