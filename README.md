# WindPower - Slimefun4 风力发电附属插件

一个为 Slimefun4 (MC 1.20) 设计的附属插件，添加了完整的风力发电科技树，包含 10 种新材料、7 个等级的风力发电机和 11 项研究。

## 要求

- Minecraft 1.20+
- Java 17+
- Slimefun4 RC-34 或更高版本
- Spigot / Paper 服务端

## 安装

1. 将 `WindPower-1.0.0.jar` 放入服务器的 `plugins/` 目录
2. 确保已安装 Slimefun4
3. 重启服务器
4. 在 Slimefun 指南中找到「风力能源」分类

## 风力发电机 (7 级)

所有发电机均使用**自定义头颅皮肤**作为材质，3D 立体外观。

| 等级 | 发电量 | 缓冲容量 | 高度限制 | 关键材料 |
|------|--------|---------|---------|---------|
| 基础 | 20 J/s | 无 | Y≥60 | 铁块+碳纤维板+强化叶片 |
| 强化 | 35 J/s | 无 | Y≥60 | 钛锭+基础涡轮 |
| 高级 | 45 J/s | 无 | **无视** | 末影之眼+钻石块+风之水晶 |
| 量子 | 150 J/s | 无 | **无视** | 量子核心+下界合金 |
| 量子MK I | 180 J/s | 1024 J | **无视** | 风暴矩阵+下界之星 |
| 量子MK II | 256 J/s | 2048 J | **无视** | 风暴矩阵+末影水晶+下界之星 |
| 量子MK III | 512 J/s | 4096 J | **无视** | 4x MK II + 时空共振核心 + 下界之星 |

## 发电机制

- **高度要求**: 基础/强化涡轮需放置在最低高度以上；高级及以上无视高度
- **天空视野**: 上方有直接天空视野时满功率发电
- **半遮挡**: 天空视野被部分遮挡时发电量减半
- **缓冲容量**: 量子以下无缓冲（发电需即时消耗，否则浪费）；量子MK I及以上可储存电量
- **全遮挡**: 完全 enclosed 时不发电
- 所有数值可在 `config.yml` 中调整

## 命令

| 命令 | 说明 | 权限 |
|------|------|------|
| `/windpower` 或 `/wp` | 查看所有发电机信息 | 默认 |
| `/windpower reload` | 重载配置 | `windpower.admin` |

## 配置

所有发电量、高度要求、容量均可通过 `config.yml` 调整：

```yaml
wind-turbines:
  basic:
    energy-output: 20
    min-height: 60
    capacity: 128
    ignore-height: false
  # ... 其他等级
  quantum-mk3:
    energy-output: 256
    min-height: 0
    capacity: 8192
    ignore-height: true
```

## 头颅纹理来源

所有发电机使用来自 minecraft-heads.com 和 mc-heads.com 的自定义头颅纹理：
- 基础: Machine Part (工业机械零件)
- 强化: Cargo Node (科技组件)
- 高级: Machine Part Alt (工业方块)
- 量子: Battery (能量存储装置)
- 量子MK I: Computer (高科技电路板)
- 量子MK II: Camera (精密光学设备)
- 量子MK III: Wind Entity (风之能量体)
